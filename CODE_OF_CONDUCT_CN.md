# 贡献者公约 · Code of Conduct

## 我们的承诺

为了营造一个开放和欢迎的环境，我们作为贡献者和维护者承诺：不论年龄、体型、明显或不明显的残疾、族裔、性别特征、性别认同和表达、经验水平、教育程度、社会经济地位、国籍、个人外貌、种族、种姓、肤色、宗教、或性取向如何，在我们的项目和社区中**每个人**都能获得无骚扰的体验。

我们承诺以**有助于构建开放、友好、多元、包容、健康社区**的方式行事和互动。

## 我们的标准

有助于创造积极环境的行为示例包括：

* 对他人表达共情与善意
* 尊重不同的观点、见解与经验
* 优雅地给出和接受建设性反馈
* 承担责任，并向受错误影响的人道歉，从经验中学习
* 不只关注个人利益，也关注整个社区的最大利益

不可接受的行为示例包括：

* 使用性别化的语言或图像，对任何人进行性关注或性挑逗
* 挑衅、侮辱或贬损的评论，人身或政治攻击
* 公开或私下骚扰
* 未经明确许可，发布他人的私人信息（如物理地址或电子邮箱）
* 其他在职业场合中可被合理认为不当的行为

## 执行责任

社区领导者有责任澄清和执行我们可接受行为的标准，并针对他们认为不当、威胁性、冒犯性或有害的任何行为采取适当和公平的纠正措施。

社区领导者有权利和责任**删除、编辑或拒绝**与本行为准则不符的评论、提交、代码、wiki 编辑、issue 和其他贡献，并在适当时候说明理由。

## 适用范围

本行为准则适用于**所有社区空间**，也适用于个人在公共空间**正式代表本社区**时。代表本社区的例子包括：使用官方电子邮箱、通过官方社交媒体账号发帖，或在在线/线下活动中作为指定代表行事。

## 举报

辱骂、骚扰或其他不可接受的行为，可通过 [GitHub Issues](https://github.com/badhope/MCP-HUB/issues) 或 `SECURITY_CN.md` 中描述的私有安全公告渠道向负责执行的社区领导者举报。

所有投诉都会被**及时、公平**地审查和调查。

所有社区领导者都有义务尊重事件举报者的隐私和安全。

## 执行指南

社区领导者在决定对违反本行为准则的行为采取何种后果时，将参考以下社区影响指南：

### 1. 纠正

**社区影响**：使用不恰当的语言或其他被视为不专业或不受欢迎的行为。

**后果**：社区领导者的私下书面警告，澄清违规的性质并解释为何该行为不当。可能会要求公开道歉。

### 2. 警告

**社区影响**：单次事件或一系列行为造成的违规。

**后果**：警告并说明持续行为的后果。在特定期限内，**不得**与相关人员互动，包括主动联系执行者。这包括在社区空间和社交媒体等外部渠道避免互动。违反这些条款可能导致临时或永久封禁。

### 3. 临时封禁

**社区影响**：严重违反社区标准，包括持续的不当行为。

**后果**：在特定期限内临时禁止与社区进行任何形式的互动或公开交流。在此期间**不得**与相关人员进行公开或私下互动，违者可能导致永久封禁。

### 4. 永久封禁

**社区影响**：表现出违反社区标准的模式，包括持续的不当行为、骚扰个人、或贬损/攻击某类人群。

**后果**：永久禁止在社区内进行任何形式的公开互动。
