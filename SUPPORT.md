## How to get help

| Kind of question | Where to ask |
|------------------|--------------|
| "How do I use feature X?" | Discussions, then Issues if no answer |
| "I think I found a bug" | Issues (use the bug template) |
| "I have an idea / feature request" | Discussions → Ideas category |
| "I want to add a new MCP server" | Issue with the `server_submission` template |
| "I found a security issue" | **See [SECURITY.md](SECURITY.md) — do not file public issues** |
| "I want to contribute code" | See [CONTRIBUTING.md](CONTRIBUTING.md) |

## Response time expectations

This is an open-source project maintained on a best-effort basis. There is
no SLA. Typical response times:

- Critical security issues: within 24 hours
- Bug reports: within 1 week
- Feature requests / questions: within 1 month, or not at all

If you need guaranteed response times, please fork the repository.

## Commercial support

Commercial support is not currently offered. If you need a feature added
or a bug fixed urgently, please open a well-described issue and consider
sponsoring development.
