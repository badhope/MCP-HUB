# `docs/internal/` — Maintainer-only documentation

This directory holds working notes and design history that are
useful for maintainers but **not intended for end users**.

| File | What it is |
|---|---|
| `AGENT_GUIDE_CN.md` | Chinese translation of the original AI-agent guide |
| `COMPLETENESS_SCORING_GUIDE.md` | Design notes for the server-completeness scoring algorithm |
| `NOTABLE_PROJECTS_GUIDE.md` | Curated upstream projects the maintainer tracks manually |

If you are an end user, you probably want the [top-level README](../../README.md)
or [docs/API.md](../../docs/API.md) instead.

> Historical documents (the rebrand changelog, the pre-rewrite
> system guide, the May 2026 improvement plan, the repo-intro
> draft, the hand-curated server catalog, and the superseded
> English `AGENT_GUIDE.md`) were pruned in the 2026-06 cleanup.
> See `CHANGELOG.md` for what each one was and why it was removed.
