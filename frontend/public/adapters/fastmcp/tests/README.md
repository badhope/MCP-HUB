## How to reproduce

Each row was generated by the same script:

```bash
# 1. Run the install script (idempotent).
curl -fsSL https://raw.githubusercontent.com/badhope/MCP-HUB/main/frontend/public/adapters/fastmcp/install.sh | bash

# 2. Drop the universal mcpServers JSON into the platform's config
#    file. (See README.md for the per-platform path.)

# 3. Boot the platform, ask it "what tools do you see?". Confirm
#    that `fastmcp.add` (or whichever example tool upstream ships)
#    shows up in the tool list. That's the test.
```

## Result interpretation

- ✅ pass: the example tool is visible in the platform's tool list
  AND a single tool call (e.g. "add 2 and 3") returns the expected
  result.
- ❌ fail: not yet seen. If a row flips to ❌, update
  `adapter.json` with the fix and bump the `last_verified` date.
